ESP Log File Parser

Please note that if you use this on large log files, it will most likely hang up your browser. In the case of large files, use something like curl. Here is an example of a call using curl:

curl -X POST -F 'espname=jake' -F 'esptype=shallow' -F 'espmode=real' -F 'logfile=@/path/to/logfile.log' http://localhost:8080/cgi-bin/log-parse.cgi
