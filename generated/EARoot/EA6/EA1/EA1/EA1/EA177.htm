This class represents the concept of a #lt;code#gt;Sample#lt;/code#gt; for MBARI's ESP. A sample is basically any mass of material that some process will be run on. It is usually in a liquid forms (lysate solution, sea water, etc.) and thus has an associated volume. If the material is not suspended in solution, the volume attributes will be empty (or null).
