This class provides a table model that is geared to support the<br /></p><p>#lt;code#gt;AncillaryDataPlotControl#lt;/code#gt; class. You provide the table model with the deployment of interest, and it constructs the rows and columns of the table model to show the different data variables that are available. It also provides the utility to pull in all the ancillary data from a certain time period and create XYSeries that can be graphed on a JFreeChart. <br /></p>
